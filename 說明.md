# 【專案名稱】
OptiMeasure_AOI

## 【角色與任務】
你是一位資深的機器視覺 (AOI) 與 UI 軟體工程師。請使用 `Python + PySide6` 以及 `OpenCV` 為我撰寫一款專業的影像特徵標註、量測與前處理軟體 `OptiMeasure_AOI`。
程式碼必須具備高內聚低耦合的物件導向架構 (MVC模式)，嚴格區分 UI 邏輯、影像處理邏輯與圖形物件 (Graphics Items) 邏輯。請提供完整且可執行的程式碼。

## 【核心技術架構】
* **UI 框架：** PySide6。主視窗使用 `QMainWindow`。影像顯示核心**必須使用 `QGraphicsView` 與 `QGraphicsScene`**，以支援向量圖形疊加與物件導向的互動。
* **影像處理：** 使用 OpenCV (`cv2`) 負責底層影像讀取、ROI 切割與影像增強演算法。
* **雙層影像架構：** 記憶體中必須維護「原始影像 (Original Image)」與「顯示影像 (Display Image)」。影像增強僅作用於顯示影像，而滑鼠探測灰階值時必須讀取原始影像的真實數值。

## 【核心功能規格細節】

### 1. 影像顯示與檢視控制 (主畫布)
* **拖曳讀圖：** 支援將圖片檔 (JPG/PNG/BMP) Drag & Drop 拖入 `QGraphicsView` 中自動載入並自適應顯示。
* **滾輪縮放：** 滑鼠滾輪可針對游標所在位置進行圖片放大與縮小。
* **模式切換與平移：** 實作狀態機。在「平移/選取模式」下，當影像放大時，左鍵點擊並拖曳可平移影像 (Pan)。
* **像素資訊即時顯示：** 滑鼠在影像上移動時，需在 StatusBar 即時顯示游標位置的「原始影像」Pixel 座標 (X, Y) 與真實灰階值 (Gray Value)（無需按壓任何按鍵）。

### 2. 影像增強前處理視窗 (Image Enhancement Dialog)
* 主介面提供一個「影像增強」Button，點擊後彈出獨立的子視窗 (`QDialog`)。
* 在子視窗內提供即時預覽影像增強效果的控制元件（僅改變主畫面的顯示影像，不動原圖）：
    * **Gamma 校正：** 提供 Slider 調整 Gamma 值（例如 0.1 ~ 3.0），拉提暗部細節。
    * **線性縮放 (對比/亮度)：** 提供 Gain 與 Offset 拉桿，調整對比與亮度。

### 3. 互動式特徵繪製與微調
* **外觀要求：** 所有繪製的圖形必須是**「空心」的（內部透明無填色，`Qt.NoBrush`）**，以免遮擋底圖。並提供 UI 讓使用者自訂線條顏色。
* **圖形個別刪除：** 處於選取狀態的圖形，可透過按下鍵盤 `Delete` 鍵、按滑鼠右鍵選單的「刪除」，或是透過數據清單來移除。
* 提供工具列切換繪圖模式，操作邏輯如下：
    * **圓形 (Circle)：** 左鍵 Click Down 決定圓心，Drag 決定半徑，Click Up 完成。
    * **正交矩形 (rectangle1)：** 角度固定為 0/90 度。左鍵 Drag 畫出對角線範圍。Focus 時可拖曳邊緣控制點改變長寬。
    * **旋轉矩形 (rectangle2)：** 帶有任意角度。左鍵 Drag 畫出初始大小。Focus 時需提供旋轉機制（如額外的旋轉把手），讓使用者能以矩形中心為軸心自由旋轉。
    * **直線 (Line)：** 左鍵 Drag 畫出起終點。
* **圖形微調：** 在「選取模式」下點擊圖形後，可直接拖曳整個圖形改變位置。支援使用鍵盤 `Up`, `Down`, `Left`, `Right` 進行像素級的位置微調。

### 4. 幾何參數輸入產生 (Parametric Input)
* 比照 Halcon 運算子邏輯，提供 UI 面板手動輸入數值精確產生圖案：
    * `gen_circle(x, y, radius)`
    * `gen_rectangle1(row1, col1, row2, col2)` 
    * `gen_rectangle2(x, y, angle, width_half, height_half)`
    * `gen_line(x1, y1, x2, y2)`

### 5. 數據清單與雙向連動 (QTableWidget)
* UI 配置一個表格 (`QTableWidget`)。
* 畫布上新增圖形時，表格同步新增一筆資料（圓形：圓心, 半徑；矩形：中心, 長, 寬, 角度；直線：起終點, 長度）。
* **雙向連動：** * 點擊表格項目 -> 畫布上對應圖形 Highlight。
    * 畫布上拖曳微調圖形 -> 表格數值即時更新。
    * 刪除表格內的項目 -> 畫布上的圖形同步消失。

### 6. 局部放大鏡工具 (Magnifier / Loupe)
* 主介面提供一個 Button，點擊後彈出獨立子視窗 (`QDialog` / `QWidget`)。
* **中心標示：** 放大鏡視窗的正中心，必須畫一個 1x1 Pixel 的高對比色（例如紅色）方框，以精確標示目前對準的像素。
* **UI 元件：** 包含「跟隨滑鼠」的 Checkbox，以及選擇放大倍率的 Combobox (100%, 150%, 200%, 300%)。
* **更新邏輯：**
    * **勾選「跟隨滑鼠」：** 主畫面滑鼠移動 (`mouseMoveEvent`) 時，擷取游標 ROI 並放大顯示。
    * **取消勾選「跟隨滑鼠」：** 只有在主畫面點擊左鍵 (`mousePressEvent`) 時，才擷取並更新放大影像。
